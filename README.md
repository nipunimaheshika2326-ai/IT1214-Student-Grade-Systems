# IT1214-Student-Grade-System
## project purpose 
 This java console application allows teachers to manage student records
 digitally.Teachers can :
   - Add a student
   - Display all students
   - Search students by student id
   - Calculate and display the average marks

## How to run     
1.Compile the java file
```bash
javac Student.java
```
2.Run the program
```bash
java StudentManagementSystem
```
3.Use the menu
Choose an option from the menu:

  i   - Add student
  
  ii  - Display all students
  
  iii - Search students by student id
  
  iv  - Calculate average marks
  
  v   - Exit
